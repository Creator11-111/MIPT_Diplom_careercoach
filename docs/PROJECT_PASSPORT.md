# Project Passport - Financial Career Coach

**Project Name:** Интеллектуальная система карьерного коучинга для финансового сектора  
**Type:** Выпускная квалификационная работа (ВКР)  
**Institution:** МФТИ  
**Date:** 2025  
**Version:** 1.0.0

---

## 1. Problem Statement

### 1.1 Проблема

В современном финансовом секторе специалисты сталкиваются с рядом вызовов при построении карьеры:

1. **Отсутствие специализированных карьерных консультантов**
   - Большинство карьерных коучей не имеют глубокого понимания специфики финансового сектора
   - Недостаточное знание финансовых инструментов, стандартов и сертификатов

2. **Сложность поиска релевантных вакансий**
   - Традиционные методы поиска (по ключевым словам) не учитывают семантическое сходство
   - Недостаточная точность поиска из-за отсутствия понимания контекста

3. **Недостаток персонализированных рекомендаций**
   - Отсутствие систем, учитывающих индивидуальный профиль, опыт и карьерные цели

4. **Сложность построения карьерной траектории**
   - Специалистам сложно самостоятельно определить необходимые навыки для перехода на желаемую позицию

### 1.2 Актуальность

- Рост числа специалистов в финансовом секторе
- Увеличение сложности требований к компетенциям
- Необходимость непрерывного обучения и развития
- Рост популярности AI-решений для карьерного консультирования

## 2. Goals and Objectives

### 2.1 Цель проекта

Разработка интеллектуальной системы карьерного коучинга для специалистов финансового сектора, использующей технологии искусственного интеллекта для персонализированного подбора вакансий, курсов и построения карьерных траекторий.

### 2.2 Задачи

1. Разработать систему интеллектуального интервью с AI-консультантом
2. Реализовать семантический поиск вакансий с использованием FAISS и LLM
3. Создать систему персонализированных рекомендаций с анализом разрывов
4. Обеспечить удобный пользовательский интерфейс

### 2.3 Критерии успеха

- ✅ Система успешно собирает профиль через естественный диалог
- ✅ Точность подбора вакансий > 80% (субъективная оценка)
- ✅ Время ответа < 5 секунд для большинства операций
- ✅ Uptime > 99% в production

## 4. User Personas

**Персона 1: Финансовый аналитик (Middle)**
- Опыт: 3-7 лет
- Цели: Переход на Senior или в инвестиционный банк

**Персона 2: Бухгалтер → CFO**
- Опыт: 10+ лет
- Цели: Карьерный рост до финансового директора

**Персона 3: Банковский специалист**
- Опыт: 2-5 лет
- Цели: Переход в финтех или инвестиции

## 5. Functional Requirements

### FR1: Интеллектуальное интервью
- AI задает 8-10 последовательных вопросов
- Вопросы адаптированы для финансового сектора
- Система уточняет полноту информации

### FR2: Построение профиля
- Извлечение структурированного профиля из диалога
- Сохранение в MongoDB
- Использование в разных сессиях

### FR3: Семантический поиск вакансий
- FAISS поиск (топ-100)
- Двухэтапная LLM фильтрация (30 → 15)
- Возврат 15 наиболее подходящих вакансий

### FR4: План развития карьеры
- Анализ разрывов между текущей и целевой позицией
- Подбор курсов для развития навыков
- Поиск промежуточных вакансий

---

## 6. Non-Functional Requirements

### Производительность
- Время ответа API < 5 секунд (95% запросов)
- FAISS поиск < 1 секунды
- Поддержка до 80 одновременных запросов

### Надежность
- Uptime > 99%
- Graceful degradation при недоступности внешних сервисов
- Автоматический retry для внешних API

### Безопасность
- Секреты в переменных окружения
- CORS настраивается через конфигурацию
- Rate limiting для защиты

## 9. Risks and Mitigations

**Риск 1: Недоступность YandexGPT API**
- Митигация: Retry логика, graceful degradation

**Риск 2: Превышение лимитов API**
- Митигация: Rate limiting, мониторинг квот

**Риск 3: Низкое качество рекомендаций**
- Митигация: Итеративное улучшение промптов, сбор обратной связи
